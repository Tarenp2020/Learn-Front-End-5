# AI Reflection

## What did you ask the AI to do?
I gave the AI context for my project and asked it to only explain the immediate thing I was asking so I could continue asking more questions step by step.

## What did it do well?
It stayed at a slow pace, and once I gave it enough context, it helped make the files easily.

## What did it get wrong or what did you have to fix?
It would say “Add this code,” but it wouldn’t tell me where to put it or whether I needed a new file. The messy organization stressed me out. I told it that anytime it talked about a file or suggested code, it needed to tell me exactly what file it belonged in and where that file was located.

## What did you learn from the AI suggestions?
Once we understood each other, it made things feel simple and less scary. Since I was transferring the code directly instead of having the AI add it automatically, I felt like I was learning how to better structure the project. If there’s an error later or I need to find and change a page the AI made, I’ll know where it is. I could ask questions make sugestions and the AI would listen answer then let me prosses before moving on.
